值迭代算法

值迭代算法（Value Iteration Algorithm）

在前文中，我们已经知道贝尔曼最优性方程（BOE）是强化学习理论的核心。现在，我们来详细介绍如何通过值迭代算法来求解BOE，进而获得最优状态价值和最优策略。

值迭代的基本思想

贝尔曼最优性方程可以写为

v=f(v)=max⁡π(rπ+γPπv)v = f(v) = \max_{\pi} (r_\pi + \gamma P_\pi v) v=f(v)=πmax​(rπ​+γPπ​v)

根据收缩映射定理，我们可以采用迭代的方法来逼近这个方程的唯一解。具体地，值迭代算法的迭代公式为

vk+1=f(vk)=max⁡π(rπ+γPπvk),k=1,2,3,…v_{k+1} = f(v_k) = \max_{\pi} (r_\pi + \gamma P_\pi v_k), \quad k = 1, 2, 3, \ldots vk+1​=f(vk​)=πmax​(rπ​+γPπ​vk​),k=1,2,3,…

其中 v0v_0v0​ 可以任意初始化。随着迭代的进行，vkv_kvk​ 会逐步收敛到最优状态价值 v∗v^*v∗，而且收敛速度是指数级的。

算法的分解与实现

虽然上面的公式看起来简洁，但在实际实现时，我们通常将其分解为两个交替进行的步骤：策略更新（policy update）和值更新（value update）。

首先，在第 kkk 次迭代时，假设我们已经有了 vkv_kvk​，我们可以通过如下方式更新策略：

πk+1=arg⁡max⁡π(rπ+γPπvk)\pi_{k+1} = \arg\max_{\pi} (r_\pi + \gamma P_\pi v_k) πk+1​=argπmax​(rπ​+γPπ​vk​)

这一步实际上是在每个状态下，选择能够使右侧最大化的策略。对于离散动作空间，这一步通常就是在每个状态下选择使 qk(a,s)q_k(a, s)qk​(a,s) 最大的动作，其中

qk(a,s)=∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vk(s′)q_k(a, s) = \sum_r p(r|s, a) r + \gamma \sum_{s'} p(s'|s, a) v_k(s') qk​(a,s)=r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vk​(s′)

接下来，利用更新后的策略 πk+1\pi_{k+1}πk+1​，我们再进行价值更新：

vk+1=rπk+1+γPπk+1vkv_{k+1} = r_{\pi_{k+1}} + \gamma P_{\pi_{k+1}} v_k vk+1​=rπk+1​​+γPπk+1​​vk​

在元素级别上，这一过程可以写为

vk+1(s)=max⁡aqk(a,s)v_{k+1}(s) = \max_a q_k(a, s) vk+1​(s)=amax​qk​(a,s)

也就是说，在每个状态 sss，我们只需计算所有可能动作的 qqq 值，然后取最大值即可。

算法伪代码

整个值迭代算法的流程可以总结如下：

  1. 初始化概率模型 p(r∣s,a)p(r|s, a)p(r∣s,a) 和 p(s′∣s,a)p(s'|s, a)p(s′∣s,a)，以及初始猜测 v0v_0v0​。
  2. 反复进行以下步骤，直到 vkv_kvk​ 收敛（即 ∥vk−vk−1∥\|v_k - v_{k-1}\|∥vk​−vk−1​∥ 小于预设阈值）：
    • 对于每个状态 sss，计算所有动作 aaa 的 qqq 值：

      qk(a,s)=∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vk(s′)q_k(a, s) = \sum_r p(r|s, a) r + \gamma \sum_{s'} p(s'|s, a) v_k(s') qk​(a,s)=r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vk​(s′)

    • 找到最大 qqq 值对应的动作 ak∗(s)a_k^*(s)ak∗​(s)，并更新策略：

      πk+1(a∣s)={1,a=ak∗(s)0,a≠ak∗(s)\pi_{k+1}(a|s) = \begin{cases} 1, & a = a_k^*(s) \\ 0, & a \neq a_k^*(s) \end{cases} πk+1​(a∣s)={1,0,​a=ak∗​(s)a=ak∗​(s)​

    • 更新状态价值：

      vk+1(s)=max⁡aqk(a,s)v_{k+1}(s) = \max_a q_k(a, s) vk+1​(s)=amax​qk​(a,s)
