贝尔曼最优公式

最优策略与贝尔曼最优性方程（BOE）

在强化学习中，我们不仅关心如何评估一个策略的好坏，更关心如何找到最优策略。最优策略的定义、存在性、唯一性以及如何求解，都是强化学习理论的核心问题。本文将围绕最优策略的定义和贝尔曼最优性方程（Bellman Optimality Equation, BOE）展开详细介绍。

什么是最优策略？

我们可以用状态价值（state value）来评价一个策略的优劣。对于任意两个策略 π1\pi_1π1​ 和 π2\pi_2π2​，如果对于所有状态 sss 都有

vπ1(s)≥vπ2(s),∀s∈Sv_{\pi_1}(s) \geq v_{\pi_2}(s), \quad \forall s \in \mathcal{S} vπ1​​(s)≥vπ2​​(s),∀s∈S

那么我们说 π1\pi_1π1​ 比 π2\pi_2π2​ 更“好”。

基于这个标准，我们可以给出最优策略的定义：

定义
如果对于所有状态 sss 以及任意其他策略 π\piπ，都有

vπ∗(s)≥vπ(s)v_{\pi^*}(s) \geq v_{\pi}(s) vπ∗​(s)≥vπ​(s)

那么策略 π∗\pi^*π∗ 就是最优策略（optimal policy）。

这个定义自然引出了许多问题：最优策略一定存在吗？最优策略是否唯一？最优策略是随机的还是确定性的？我们又该如何找到最优策略？这些问题的答案，都与贝尔曼最优性方程密切相关。

贝尔曼最优性方程（BOE）简介

要想回答上述问题，我们需要引入贝尔曼最优性方程。贝尔曼最优性方程为我们描述了最优状态价值和最优策略之间的关系

元素形式（Elementwise Form）

贝尔曼最优性方程的元素形式如下：

v(s)=max⁡π∑aπ(a∣s)(∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)v(s′)),s∈Sv(s) = \max_{\pi} \sum_a \pi(a|s) \left( \sum_r p(r|s,a) r + \gamma \sum_{s'} p(s'|s,a) v(s') \right), \quad s \in \mathcal{S} v(s)=πmax​a∑​π(a∣s)(r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)v(s′)),s∈S

也可以写成：

v(s)=max⁡π∑aπ(a∣s)q(s,a),s∈Sv(s) = \max_{\pi} \sum_a \pi(a|s) q(s, a), \quad s \in \mathcal{S} v(s)=πmax​a∑​π(a∣s)q(s,a),s∈S

其中 q(s,a)q(s, a)q(s,a) 是动作价值函数。

在这个方程中，p(r∣s,a)p(r|s,a)p(r∣s,a)、p(s′∣s,a)p(s'|s,a)p(s′∣s,a)、rrr、γ\gammaγ 都是已知的环境参数，而 v(s)v(s)v(s)、v(s′)v(s')v(s′) 是未知的，需要我们求解。π(a∣s)\pi(a|s)π(a∣s) 也可能是已知或未知，这取决于我们是否已经确定了策略。

矩阵-向量形式（Matrix-Vector Form）

将所有状态的贝尔曼最优性方程写成矩阵-向量形式，可以得到：

v=max⁡π(rπ+γPπv)\mathbf{v} = \max_{\pi} \left( \mathbf{r}_\pi + \gamma P_\pi \mathbf{v} \right) v=πmax​(rπ​+γPπ​v)

其中，

  • rπ\mathbf{r}_\pirπ​ 的每个元素为

    [rπ]s=∑aπ(a∣s)∑rp(r∣s,a)r[\mathbf{r}_\pi]_s = \sum_a \pi(a|s) \sum_r p(r|s,a) r [rπ​]s​=a∑​π(a∣s)r∑​p(r∣s,a)r

  • PπP_\piPπ​ 的每个元素为

    [Pπ]s,s′=∑aπ(a∣s)p(s′∣s,a)[P_\pi]_{s,s'} = \sum_a \pi(a|s) p(s'|s,a) [Pπ​]s,s′​=a∑​π(a∣s)p(s′∣s,a)

这里的 max⁡π\max_\pimaxπ​ 是对每个状态分量分别取最大值，即

max⁡π[∗⋮∗]=[max⁡π(s1)∗⋮max⁡π(sn)∗]\max_\pi \begin{bmatrix} * \\ \vdots \\ * \end{bmatrix} = \begin{bmatrix} \max_{\pi(s_1)} * \\ \vdots \\ \max_{\pi(s_n)} * \end{bmatrix} πmax​​∗⋮∗​​=​maxπ(s1​)​∗⋮maxπ(sn​)​∗​​

BOE的优雅与挑战

贝尔曼最优性方程（BOE）既优雅又充满挑战。它优雅之处在于用一个简洁的方程同时描述了最优策略和最优状态价值之间的关系；而挑战在于，方程右侧的最大化操作并不总是容易直接计算。

BOE的研究涉及一系列重要问题：

  • 算法：如何有效地求解这个方程？
  • 存在性：这个方程一定有解吗？
  • 唯一性：解是否唯一？
  • 最优性：这个方程的解与最优策略之间的关系是什么？

BOE的预备知识

在深入求解贝尔曼最优性方程（BOE）之前，我们需要理解几个重要的数学工具和思想，包括右侧的最大化操作、如何将BOE重写为 v=f(v)v = f(v)v=f(v) 的形式，以及收缩映射定理（Contraction Mapping Theorem）。

贝尔曼最优性方程的本质在于右侧的最大化操作。其元素形式为：

v(s)=max⁡π∑aπ(a∣s)(∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)v(s′)),∀s∈Sv(s) = \max_{\pi} \sum_a \pi(a|s) \left( \sum_r p(r|s,a) r + \gamma \sum_{s'} p(s'|s,a) v(s') \right), \quad \forall s \in \mathcal{S} v(s)=πmax​a∑​π(a∣s)(r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)v(s′)),∀s∈S

而矩阵-向量形式则为：

例子：如何从一个方程中解出两个未知数

考虑如下方程，要求解 x,a∈Rx, a \in \mathbb{R}x,a∈R：

由此可见，max⁡π∑aπ(a∣s)q(s,a)\max_\pi \sum_a \pi(a|s)q(s,a)maxπ​∑a​π(a∣s)q(s,a) 的最优性在于 π(a∗∣s)=1\pi(a^*|s) = 1π(a∗∣s)=1，其余为0，其中 a∗=arg⁡max⁡aq(s,a)a^* = \arg\max_a q(s,a)a∗=argmaxa​q(s,a)。

最优策略在于每个状态下选择使 q(s,a)q(s, a)q(s,a) 最大的动作。

BOE重写为 v=f(v)v = f(v)v=f(v) 的形式

我们可以将贝尔曼最优性方程写成自映射的形式：

v=max⁡π(rπ+γPπv)\mathbf{v} = \max_{\pi} \left( \mathbf{r}_\pi + \gamma P_\pi \mathbf{v} \right) v=πmax​(rπ​+γPπ​v)

这个方程看起来很简单，但如何求解呢？这就需要用到收缩映射定理。

收缩映射定理（Contraction Mapping Theorem）

在数学中，收缩映射定理是分析自映射方程 x=f(x)x = f(x)x=f(x) 是否有唯一解的有力工具。

1. 基本概念

  • 不动点（Fixed point）：x∈Xx \in Xx∈X 是 f:X→Xf: X \to Xf:X→X 的不动点，如果 f(x)=xf(x) = xf(x)=x。
  • 收缩映射（Contraction mapping）：fff 是收缩映射，如果存在 γ∈(0,1)\gamma \in (0, 1)γ∈(0,1)，使得

    ∥f(x1)−f(x2)∥≤γ∥x1−x2∥\|f(x_1) - f(x_2)\| \leq \gamma \|x_1 - x_2\| ∥f(x1​)−f(x2​)∥≤γ∥x1​−x2​∥

    这里的 ∥⋅∥\|\cdot\|∥⋅∥ 可以是任意向量范数。

γ\gammaγ 必须严格小于1，这样才能保证如 γk→0\gamma^k \to 0γk→0 随 k→∞k \to \inftyk→∞。

例子：

  • x=f(x)=0.5x,  x∈Rx = f(x) = 0.5x, \; x \in \mathbb{R}x=f(x)=0.5x,x∈R
    很容易验证 x=0x=0x=0 是不动点，并且 f(x)f(x)f(x) 是收缩映射，因为

    ∣0.5x1−0.5x2∣=0.5∣x1−x2∣≤γ∣x1−x2∣,∀γ∈[0.5,1)|0.5x_1 - 0.5x_2| = 0.5|x_1 - x_2| \leq \gamma|x_1 - x_2|, \quad \forall \gamma \in [0.5, 1) ∣0.5x1​−0.5x2​∣=0.5∣x1​−x2​∣≤γ∣x1​−x2​∣,∀γ∈[0.5,1)

  • x=f(x)=Ax,  x∈Rn,∥A∥<1x = f(x) = Ax, \; x \in \mathbb{R}^n, \|A\| < 1x=f(x)=Ax,x∈Rn,∥A∥<1
    x=0x=0x=0 是唯一不动点，且

2. 收缩映射定理内容

定理（收缩映射定理）
对于任何形如 x=f(x)x = f(x)x=f(x) 的方程，如果 fff 是收缩映射，则：

  • 存在性：存在唯一的不动点 x∗x^*x∗，使 f(x∗)=x∗f(x^*) = x^*f(x∗)=x∗。
  • 唯一性：不动点 x∗x^*x∗ 是唯一的。

BOE与最优性

最优性原理

假设 v∗v^*v∗ 是贝尔曼最优性方程的唯一解，即

v∗=max⁡π(rπ+γPπv∗)v^* = \max_{\pi}(r_\pi + \gamma P_\pi v^*) v∗=πmax​(rπ​+γPπ​v∗)

也就是说，π∗\pi^*π∗ 是一个策略，v∗=vπ∗v^* = v_{\pi^*}v∗=vπ∗​ 是其对应的状态价值函数。

那么，π∗\pi^*π∗ 是不是最优策略？v∗v^*v∗ 是不是所有策略中能达到的最大状态价值？

最优性定理

定理（策略最优性）
假设 v∗v^*v∗ 是 v=max⁡π(rπ+γPπv)v = \max_{\pi}(r_\pi + \gamma P_\pi v)v=maxπ​(rπ​+γPπ​v) 的唯一解，vπv_\pivπ​ 是任意策略 π\piπ 的状态价值函数，满足 vπ=rπ+γPπvπv_\pi = r_\pi + \gamma P_\pi v_\pivπ​=rπ​+γPπ​vπ​，则

换句话说，v∗v^*v∗ 就是所有策略中能达到的最大状态价值。

这就是我们研究BOE的根本原因：它描述了最优状态价值和最优策略。

分析最优策略

决定最优状态价值和最优策略的因素

最优状态价值和最优策略由哪些因素决定？从贝尔曼最优性方程（BOE）可以清楚地看出：

v(s)=max⁡π∑aπ(a∣s)(∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)v(s′))v(s) = \max_{\pi} \sum_a \pi(a|s) \left( \sum_r p(r|s,a) r + \gamma \sum_{s'} p(s'|s,a) v(s') \right) v(s)=πmax​a∑​π(a∣s)(r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)v(s′))

主要有三大因素影响最优解：

  • 系统模型：p(s′∣s,a)p(s'|s,a)p(s′∣s,a) 和 p(r∣s,a)p(r|s,a)p(r∣s,a)，即环境的状态转移概率和奖励分布。
  • 奖励设计：rrr，即每一步的即时奖励。
  • 折扣因子：γ\gammaγ，决定了未来奖励的权重。

接下来我们通过具体例子，展示 rrr 和 γ\gammaγ 如何影响最优策略。

折扣因子 γ\gammaγ 的影响

  • γ=0.9\gamma = 0.9γ=0.9 时

​ 当 γ=0.9\gamma = 0.9γ=0.9，rboundary=rforbidden=−1r_{\text{boundary}} = r_{\text{forbidden}} = -1rboundary​=rforbidden​=−1，rtarget=1r_{\text{target}} = 1rtarget​=1，策略会“敢于冒险”，甚至会进入禁区以追求更高的长期回报。

  • γ=0.5\gamma = 0.5γ=0.5 时

​ 如果将折扣因子降低为 γ=0.5\gamma = 0.5γ=0.5，其他参数不变，最优策略变得“短视”，会尽量避开所有禁区。

  • γ=0\gamma = 0γ=0 时

​ 如果 γ=0\gamma = 0γ=0，最优策略极度短视，只选择能获得最大即时奖励的动作，无法到达目标状态。

奖励设计的影响

如果我们加大进入禁区的惩罚，比如将 rforbiddenr_{\text{forbidden}}rforbidden​ 从 −1-1−1 改为 −10-10−10，最优策略也会主动避开禁区。

此外，如果我们对奖励做仿射变换（r→ar+br \to ar + br→ar+b），例如：

  • rboundary=rforbidden=−1,  rtarget=1,  rotherstep=0r_{\text{boundary}} = r_{\text{forbidden}} = -1, \; r_{\text{target}} = 1, \; r_{\text{otherstep}} = 0rboundary​=rforbidden​=−1,rtarget​=1,rotherstep​=0
  • 变为 rboundary=rforbidden=0,  rtarget=2,  rotherstep=1r_{\text{boundary}} = r_{\text{forbidden}} = 0, \; r_{\text{target}} = 2, \; r_{\text{otherstep}} = 1rboundary​=rforbidden​=0,rtarget​=2,rotherstep​=1

最优策略不会改变。这说明，最优策略只与奖励的相对大小有关，而与绝对值无关。

定理（最优策略不变性）
如果对所有奖励 rrr 做仿射变换 ar+bar + bar+b，则最优状态价值 v∗v^*v∗ 也会做仿射变换

v′=av∗+b1−γ1v' = a v^* + \frac{b}{1-\gamma} \mathbf{1} v′=av∗+1−γb​1

但最优策略保持不变。

为什么最优策略不会选择无意义的绕路？

如果对绕路不惩罚（rotherstep=0r_{\text{otherstep}} = 0rotherstep​=0），为什么最优策略不会选择无意义的绕路？

其实，折扣因子 γ\gammaγ 本身就起到了惩罚作用。例如：

  • 策略(a)：直接到达目标，回报为 1+γ1+γ21+⋯=1/(1−γ)1 + \gamma 1 + \gamma^2 1 + \cdots = 1/(1-\gamma)1+γ1+γ21+⋯=1/(1−γ)
  • 策略(b)：绕路一步后到达目标，回报为 0+γ0+γ21+γ31+⋯=γ2/(1−γ)0 + \gamma 0 + \gamma^2 1 + \gamma^3 1 + \cdots = \gamma^2/(1-\gamma)0+γ0+γ21+γ31+⋯=γ2/(1−γ)

显然，策略(a)的回报更高，最优策略自然不会选择绕路。

总结

贝尔曼最优性方程（BOE）为我们提供了分析和求解最优策略的理论基础。通过BOE，我们可以清楚地看到，最优策略和最优状态价值由环境模型、奖励设计和折扣因子共同决定。奖励的相对大小和折扣因子的设置，都会显著影响最终的策略选择。
