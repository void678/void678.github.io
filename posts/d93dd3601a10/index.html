Bellman方程的推导与直观理解

在强化学习中，状态价值（state value）是衡量某一状态“好坏”的核心指标，但如何计算状态价值？这正是Bellman方程要解决的问题。Bellman方程描述了不同状态价值之间的关系，是理解和求解最优策略的基础。

从回报的定义到递推公式

我们以一个随机轨迹为例，假设智能体在时刻 ttt 处于状态 StS_tSt​，采取动作 AtA_tAt​，获得奖励 Rt+1R_{t+1}Rt+1​，进入下一个状态 St+1S_{t+1}St+1​，如此反复。回报 GtG_tGt​ 定义为从当前时刻开始，未来所有奖励的折现和：

Gt=Rt+1+γRt+2+γ2Rt+3+⋯G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots Gt​=Rt+1​+γRt+2​+γ2Rt+3​+⋯

其中，γ\gammaγ 是折扣因子，控制未来奖励的影响力。我们可以将回报递归地表示为：

Gt=Rt+1+γGt+1G_t = R_{t+1} + \gamma G_{t+1} Gt​=Rt+1​+γGt+1​

状态价值函数 vπ(s)v_\pi(s)vπ​(s) 定义为：在策略 π\piπ 下，智能体从状态 sss 出发，期望获得的回报：

vπ(s)=E[Gt∣St=s]v_\pi(s) = \mathbb{E}[G_t \mid S_t = s] vπ​(s)=E[Gt​∣St​=s]

所以上述式子也就是

vπ(s)=Eπ[Gt∣St=s]v_\pi(s) = \mathbb{E_\pi}[G_t \mid S_t = s] vπ​(s)=Eπ​[Gt​∣St​=s]

结合回报的递推公式，有：

vπ(s)=E[Rt+1+γGt+1∣St=s]=E[Rt+1∣St=s]+γE[Gt+1∣St=s]\begin{align*} v_\pi(s) &= \mathbb{E}[R_{t+1} + \gamma G_{t+1} \mid S_t = s] \\ &= \mathbb{E}[R_{t+1} \mid S_t = s] + \gamma \mathbb{E}[G_{t+1} \mid S_t = s] \end{align*} vπ​(s)​=E[Rt+1​+γGt+1​∣St​=s]=E[Rt+1​∣St​=s]+γE[Gt+1​∣St​=s]​

进一步展开，第一项是对下一个奖励的期望，第二项是对下一个状态价值的期望。

期望的展开

E[Rt+1∣St=s]\mathbb{E}[R_{t+1} \mid S_t = s]E[Rt+1​∣St​=s]

对于第一项 E[Rt+1∣St=s]\mathbb{E}[R_{t+1} \mid S_t = s]E[Rt+1​∣St​=s]，我们可以按动作和奖励分布展开：

E[Rt+1∣St=s]=∑aπ(a∣s)∑rp(r∣s,a)r\mathbb{E}[R_{t+1} \mid S_t = s] = \sum_a \pi(a|s) \sum_r p(r|s,a) r E[Rt+1​∣St​=s]=a∑​π(a∣s)r∑​p(r∣s,a)r

这表示在状态 sss 下，按照策略 π\piπ 选择动作 aaa，获得奖励 rrr 的期望值。

解释

  • E[Rt+1∣St=s]\mathbb{E}[R_{t+1} \mid S_t = s]E[Rt+1​∣St​=s] 表示：在状态 sss 下，按照策略 π\piπ，下一步获得奖励 Rt+1R_{t+1}Rt+1​ 的期望值（平均值）。

  • π(a∣s)\pi(a|s)π(a∣s) 是策略，表示在状态 sss 下选择动作 aaa 的概率。

  • p(r∣s,a)p(r|s,a)p(r∣s,a) 是奖励分布，表示在状态 sss 下采取动作 aaa 后，获得奖励 rrr 的概率。

推导思路

  1. 首先，智能体在状态 sss 下，可能会按照策略 π\piπ 以不同概率选择不同的动作 aaa。

  2. 对于每个动作 aaa，环境又可能以不同概率 p(r∣s,a)p(r|s,a)p(r∣s,a) 给出不同的奖励 rrr。

  3. 所以，Rt+1R_{t+1}Rt+1​ 的期望值就是：对所有可能的动作 aaa，加权（π(a∣s)\pi(a|s)π(a∣s)）所有可能奖励 rrr 的加权平均（p(r∣s,a)p(r|s,a)p(r∣s,a)），再乘以奖励 rrr 本身。

直观理解

你可以把它理解为“在状态 sss 下，所有可能采取的动作和所有可能获得的奖励的加权平均”。这就是即时奖励的平均值。

E[Gt+1∣St=s]\mathbb{E}[G_{t+1} \mid S_t = s]E[Gt+1​∣St​=s]

展开为：

E[Gt+1∣St=s]=∑s′E[Gt+1∣St=s,St+1=s′] p(s′∣s)=∑s′E[Gt+1∣St+1=s′] p(s′∣s)=∑s′vπ(s′) p(s′∣s)=∑s′vπ(s′)∑ap(s′∣s,a)π(a∣s)\begin{align*} \mathbb{E}[G_{t+1} \mid S_t = s] &= \sum_{s'} \mathbb{E}[G_{t+1} \mid S_t = s, S_{t+1} = s'] \, p(s' \mid s) \\\\ &= \sum_{s'} \mathbb{E}[G_{t+1} \mid S_{t+1} = s'] \, p(s' \mid s) \\\\ &= \sum_{s'} v_\pi(s') \, p(s' \mid s) \\\\ &= \sum_{s'} v_\pi(s') \sum_a p(s' \mid s, a) \pi(a \mid s) \end{align*} E[Gt+1​∣St​=s]​=s′∑​E[Gt+1​∣St​=s,St+1​=s′]p(s′∣s)=s′∑​E[Gt+1​∣St+1​=s′]p(s′∣s)=s′∑​vπ​(s′)p(s′∣s)=s′∑​vπ​(s′)a∑​p(s′∣s,a)π(a∣s)​

解释

  • Eπ[Gt+1∣St=s]\mathbb{E}_\pi[G_{t+1} \mid S_t = s]Eπ​[Gt+1​∣St​=s] 表示：在状态 sss 下，下一步开始的回报 Gt+1G_{t+1}Gt+1​ 的期望值。

  • p(s′∣s,a)p(s'|s,a)p(s′∣s,a) 是转移概率，表示在状态 sss 下采取动作 aaa 后，转移到下一个状态 s′s's′ 的概率。

  • vπ(s′)v_\pi(s')vπ​(s′) 是下一个状态 s′s's′ 的价值。

推导思路

  1. 在状态 sss 下，智能体会以概率 π(a∣s)\pi(a|s)π(a∣s) 选择动作 aaa。
  2. 采取动作 aaa 后，环境会以概率 p(s′∣s,a)p(s'|s,a)p(s′∣s,a) 转移到下一个状态 s′s's′。
  3. 到达 s′s's′ 后，未来的期望回报就是 vπ(s′)v_\pi(s')vπ​(s′)。
  4. 所以，Gt+1G_{t+1}Gt+1​ 的期望值就是对所有可能的 aaa 和 s′s's′，加权 vπ(s′)v_\pi(s')vπ​(s′) 的平均值。

直观理解

这部分可以理解为“在状态 sss 下，考虑所有可能采取的动作和所有可能到达的下一个状态，未来回报的加权平均”。这就是未来奖励（下一步的累积折扣奖励）的平均值。

Bellman方程的最终形式

将上述两部分合并，得到Bellman方程的完整表达：

vπ(s)=∑aπ(a∣s)[∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vπ(s′)]v_\pi(s) = \sum_a \pi(a|s) \left[ \sum_r p(r|s,a) r + \gamma \sum_{s'} p(s'|s,a) v_\pi(s') \right] vπ​(s)=a∑​π(a∣s)[r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vπ​(s′)]

这就是著名的Bellman方程。它揭示了每个状态的价值由两部分组成：一是当前动作带来的即时奖励，二是未来状态的价值（折现后）。每个状态都对应这样一条方程，所有状态的价值函数共同构成一个方程组。

这不是一个式子！！如果我们有 n 个状态，那就有 n 个这样的式子

例题：方格世界中的Bellman方程

以一个简单的4格世界为例，假设策略是确定性的。我们可以直接将已知的转移概率、奖励和策略代入Bellman方程，逐步求解每个状态的价值。

vπ(s)=∑aπ(a∣s)[∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vπ(s′)]v_\pi(s) = \sum_a \pi(a|s) \left[ \sum_r p(r|s,a) r + \gamma \sum_{s'} p(s'|s,a) v_\pi(s') \right] vπ​(s)=a∑​π(a∣s)[r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vπ​(s′)]

例如，假设 s1s_1s1​ 的唯一动作会转移到 s3s_3s3​，且奖励为0，则有：

vπ(s1)=0+γvπ(s3)v_\pi(s_1) = 0 + \gamma v_\pi(s_3) vπ​(s1​)=0+γvπ​(s3​)

同理，其他状态也可以写出类似的方程。通过递推或代入法，可以从最后一个状态开始，逐步求解出所有状态的价值。如果折扣因子 γ=0.9\gamma = 0.9γ=0.9，最终可以得到：

vπ(s4)=10,vπ(s3)=10,vπ(s2)=10,vπ(s1)=9v_\pi(s_4) = 10, \quad v_\pi(s_3) = 10, \quad v_\pi(s_2) = 10, \quad v_\pi(s_1) = 9 vπ​(s4​)=10,vπ​(s3​)=10,vπ​(s2​)=10,vπ​(s1​)=9

随机策略下的Bellman方程

如果策略或环境是随机的，比如 s1s_1s1​ 有一半概率获得奖励0转移到 s3s_3s3​，一半概率获得奖励-1转移到 s2s_2s2​，则Bellman方程变为：

vπ(s1)=0.5[0+γvπ(s3)]+0.5[−1+γvπ(s2)]v_\pi(s_1) = 0.5[0 + \gamma v_\pi(s_3)] + 0.5[-1 + \gamma v_\pi(s_2)] vπ​(s1​)=0.5[0+γvπ​(s3​)]+0.5[−1+γvπ​(s2​)]

同样可以列出其他状态的方程，最终通过代入法求解。可以发现，策略不同，状态价值也会发生变化，这正是Bellman方程在策略评估中的作用。
